# SunnysideUp Landing Page

A responsive landing page for the Sunnyside Agency, created as part of Frontend Challenges. This project is built with **HTML**, **CSS**, and **JavaScript**, focusing on clean, modern design and responsive user experience.

## Project Overview

SunnysideUp is a landing page designed to showcase a fictional agency's services. The goal of this project is to practice building a fully responsive layout from a design mockup, implement interactive UI elements, and structure HTML/CSS for maintainability.

## Features

- Fully responsive layout for desktop and mobile devices  
- Mobile navigation menu with toggle functionality  
- Semantic HTML structure for accessibility and SEO  
- Modern styling using CSS Flexbox and Grid  
- Smooth hover effects and transitions  
- Sectioned layout for hero, features, testimonials, and call-to-action  

## Technologies Used

- **HTML5** – Semantic structure and content layout  
- **CSS3** – Flexbox and Grid for responsive design, custom styling  
- **JavaScript** – Interactive mobile navigation menu and basic UI behavior  

## Design Goals

- Build a pixel-perfect replica of the provided Frontend Challenges design  
- Ensure mobile-first responsiveness  
- Apply reusable CSS classes for maintainability  
- Focus on accessibility standards (aria labels, proper headings)
